# Projeto TCS - WAR

## Autor
Layne Alves

## Objetivo
Este projeto em C demonstra o uso de structs para cadastrar e exibir múltiplos territórios.  
Cada território possui:

- Nome do território
- Cor do exército
- Quantidade de tropas

O programa solicita ao usuário que cadastre 5 territórios e, em seguida, exibe todos os dados cadastrados.

## Tecnologias
- Linguagem: C
- Compilador recomendado: GCC
- Ambiente de desenvolvimento: VS Code (ou similar)

## Como compilar e executar

1. Abra o terminal na pasta onde está o arquivo `territorio.c`
2. Compile o programa:

```bash
gcc territorio.c -o territorio
